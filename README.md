# 계산기

## 기능

-   값 입력
    -   숫자, 연산자, 초기화 등을 입력하면 받아오는 기능
-   계산기능
    -   더하기, 빼기, 곱하기, 나누기, 제곱, 제곱근
-   기타기능
    -   소숫점, 양수/음수, 초기화

---

##

-   초기 값
    -   현재 계산중인 반환 값
    -   화면에 표시되는 값
    -   양수, 음수 상태 값(네거티브)
